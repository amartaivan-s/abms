Simple Diffusion Model

In this demo, we simulate a diffusion process where agents can represent the spread of an innovation, virus infection, or social news. Initially, a set of agents are marked as infected while others are non-infected. Each time step:

  • Agents move randomly on a grid.
  • When an infected agent interacts with a non-infected one, the latter‚Äôs infection level increases.
  • Once the infection level passes a threshold, the agent becomes infected.

By changing initial values (grid size, number of infected, infection rate), we can investigate how diffusion dynamics unfold.

About Mesa

This model was built with Mesa, a Python framework for Agent-Based Modeling. Mesa provides tools to build, visualize, and analyze ABMs directly in Python and runs in the browser for easy demonstration.

The version used here is Mesa 2.4, which retains the classic CanvasGrid and ChartModule visualization components (later replaced in Mesa 3.x).
